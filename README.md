# tap.

Offline party game for 2 to 5 players.

Players place their fingers on one screen, the app picks one person at random,
and the chosen player gets a hidden challenge or a custom task.

Current platforms:
- Flutter mobile app
- iOS project
- Android project

Android release signing:
- Copy `key.properties.example` to `key.properties`
- Fill in your real keystore values
- Keep the keystore file out of git
